# 📖 ChatInterface <a name="about-project"></a>

**Jobs API** is a simple Node.js REST API that allows a job seeker to keep track of their job applications and the various interviewing stages.

## 🛠 Built With <a name="built-with">Node.js</a>

### Tech Stack <a name="tech-stack"></a>

- MongoDB

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- Node installed on your computer

### Setup

Clone this repository to your desired folder:

- git clone https://github.com/willy836/jobs-api.git
- cd jobs-api/

### Install

Install this project with:

- run **npm install** to install necessary dependencies

### Usage

To run the project on your locally, execute the following command:

- npm run start
- On your browser, type http://127.0.0.1:3000 to see the app
